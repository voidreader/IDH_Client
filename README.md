# IDH_Client
'이런 영웅은 싫어' 모바일 게임 유니티 클라이언트 프로젝트

## 📱 프로젝트 소개
모바일 RPG 게임 "이런 영웅은 싫어"의 클라이언트입니다.
단독으로 실행 할 수 없으며 IDH_Node와 연계되어야 합니다. 


### ⚙️ 클라이언트 개발 환경
- `Unity 2018.4.17f1`
- `C#`
- `AssetBundle`
- `NGUI`
- `Spine`


### 🔨 폴더 구조
- ani	연출 애니메이션 데이터
- AssetBundleManager	에셋번들 플러그인의 자체 에셋번들 다운로드 기능 사용시 필요한 URL
- AssetBundleResources	에셋번들링 될 대상 파일들
- AssetBundleTest	실제로 사용되는 에셋번들 다운로드 기능(AssetLoader.cs)의 로컬 테스트용 폴더
- Editor	에셋번들 플러그인 관련 에디터 파일
- GachaAni	연출 효과 관련 리소스 파일 (Prefabs, Materials, Animations 등)
- GamePot	게임팟 플러그인
- GrapicResources	마이룸 시스템에서 사용되는 리소스
- Plugins	플랫폼별 폴더와 기타 플러그인 에셋들
- Prefabs	프리팹 파일들
- Scenes	씬 파일 및 관련 파일
- scripts	cs 스크립트 파일
- shaders	NGUI용 스탠실마스킹용 쉐이더
- sprites	로컬에서 사용되는 이미지 파일
- TestFolder	튜토리얼용 부분강조효과용쉐이더
